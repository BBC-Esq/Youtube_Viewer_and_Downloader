# Youtube Downloader
Provides a nice graphical user interface to use the [pytubefix library](https://github.com/JuanBindez/pytubefix) to download files from Youtube.

<img width="1112" height="653" alt="image" src="https://github.com/user-attachments/assets/0e50a807-96cf-42a7-8fd6-6c24b7b0ca34" />

## Installation

Create Virtual Environment
```
python -m venv .
```
Activate
```
.\Scripts\activate
```
Install Libraries
```
python install.py
```

## Usage

1) Activate your virtual environment
2) Run the script.
```
python main.py
```
